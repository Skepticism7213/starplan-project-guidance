# StarPlan Project Guidance

This repository contains the planning and handoff materials for **StarPlan Loop**, a Qwen-callable astronomy training Skills package for Challenge Cup Track 3, Direction 3: “星语·面向 AI 的天文实训”.

The current direction is a reproducible loop for campus astronomy activities:

```text
observation request
-> deterministic astronomy calculation
-> observation plan and outreach pack
-> observation log
-> evidence-based review
-> revised next plan
```

## Contents

- [`starplan-project-guidance/`](starplan-project-guidance/) - project plan, competition enhancements, Qwen startup prompt, transfer log, kickoff report, and alignment notes.
- [`AGENTS.md`](AGENTS.md) - repository conventions and document source-of-truth rules.

The repository currently records planning materials only. It does not contain the implementation of the Skills package.
